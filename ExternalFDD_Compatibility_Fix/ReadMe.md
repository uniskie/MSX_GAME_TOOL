# External FDD Compatibility Fix

内蔵FDD以外から起動できないゲームのためのツールです。

きゃんきゃんバニースペリオール用ですが、  
他のゲームももしかしたら有効かもしれません（未確認）

## 使用方法

1. extFdd_fix.dskを起動
2. "INSERT DISK AND HIT ANY KEY"  
   と表示されたらゲームのディスクを入れて  
   SPACEキーやRETURNキーなどを押下

- きゃんきゃんバニースペリオールなど  
  CTRLキーを押しながら起動する必要のあるものは  
  "INSERT DISK AND HIT ANY KEY"がでるまで  
  電源投入時からCTRLキーを押し続ける

## 原因

ブートセクタのCE1EHが呼ばれたときに  
DEレジスタにはいっている  
「ページ1がRAMに切り換わるアドレス(F36BH)」について  
外付けだと機能しないケースがあり、  
このルーチンを使用してページ1がRAMになることを期待した  
プログラムがうまく動作しない問題へ力技で対処するものです。

## 対処

ページ1をRAMに変更してからキー入力を待ち、  
何かキーが押されたら  
ゲームのブートセクタを読み込み  
ゲーム側の処理へジャンプします。  

## 動作条件

PHYDIOを呼ぶのでDOS1専用です。
